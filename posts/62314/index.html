如何导出微信读书笔记？

发表于 2021-03-06 | 分类于 阅读   |   全文共887字   |   阅读时长3分钟

只要开始做事，一个人就要面临做错、做不好的风险。

据我了解，电子书阅读体验比较好的产品，首当其冲应该是微信读书吧。

在 2019年 也分享过《关于微信读书你可能不知道的 56 条阅读指南》，虽然产品后面有更新，但还是有一些参考价值的。

我的主要电子书阅读产品有 2 个：以 得到电子书 为主，微信读书为辅。

因为得到的书相对纯粹一点，微信读书有不少小说的内容推荐，加上，最近又有一些纸质书的“干扰”；但是微信读书支持导入本地电子书，对于阅读已经下架或者不可言喻的书，是一个不错的选择。

微信读书虽然现在没作为主力，刚看了一下，在 2019 年里差不多有半年时间都在用，也用它读完了 92本书，花了 596 小时。今天和大家分享一下，导出微信读书 App 笔记的三种方法。

方法一：采用浏览器插件

插件名称：微信读书笔记助手

安装方法：Chrome 商店搜索名字 或者 在 Github 下载 crx 文件安装

核心功能是以 markdown 的形式导出微信读书的笔记；支持一键导出章节的全部标注、热门标注、想法、书的目录，格式可以自定义；支持导出标注的图片、划线等；更多的可以自己挖掘下。

使用方法：安装插件 - 登录网页版微信读书 - 打开一本书 - 使用插件（如上图）；

它的好处是，对单个章节、单本书的笔记、想法等处理比较友好；其次 markdown 格式很方便，可以任意迁移；没想到还支持划线图片导出，这是其他产品没有的。

方法二：直接同步到 flomo

flomo 作为一个碎片化输入的产品，我目前主要用它来记录一些零碎的信息。读书的笔记、想法都是其中一种；详情可读《我是如何解决信息输入的3大问题？》。

flomo 本身不能同步，但是提供了 API，所以 @EXTRASTU 可以通过插件的方式实现微信读书笔记。

插件名称：flomoplus

安装地址：Chrome 商店搜索名字 或者 在 Github 下载 crx 文件安装

核心功能：支持导入微信读书（含公众号）笔记、微博、即刻动态、得到电子书笔记等

使用方法：安装插件 - 在插件设置好 API 地址 - 导入微信读书 - 扫码登录微信；

它的好处是：不需要更多的操作，可以直接导入 flomo ，通过标签，亦可进行归类；主要还是方便。

方法三：使用 Klib 导入所有笔记

工具名称：Klib

下载地址：App Stroe 搜索名字 或者 在Klib官网下载；

核心功能：支持微信读书（含公众号）笔记、Kindle、Apple Books等笔记导入和管理

使用方法：下载软件 - 导入微信读书笔记 - 扫码登录 ；然后就全部导入了。

它的好处是：不仅仅有笔记导入，还有管理，例如编辑、复制、同步到印象笔记等。

除此之外，微信读书本身也支持单本书笔记导出，对比之下，效率较低，不够灵活。

以上就是我目前处理微信读书笔记的三种方式，希望对你有点点点点用。
